Конвертер для преобразования из std::vector в list

template<typename T>
struct cppvector_to_list
{
    static PyObject *convert( T const &v )
    {
        boost::python::list list;

        for ( auto const &it : v )
        {
            list.append( *it );
        }

        return incref( list.ptr() );
    }
};

Сборка Boost.python с помощью MinGW

Ср. 13 Май 2015 Tags Python / Boost
  1. Скачать архив с сайта
  2. Распаковать в C:\
  3. Перейти в папку C:\boost_1_xx_x
  4. Выполнить:
bootstrap.bat mingw
b2 --build-type=complete --with-python stage
  1. В папке C:\boost_1_xx_x\stage\lib будут лежать собранные .dll

    Расшифрофка имен файлов приведена здесь

Для сборки x64 нужно использовать ключ address-model=64

b2 --build-type=complete address-model=64 --with-python ...
